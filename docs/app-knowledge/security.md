# Sicurezza di APPecchio

## Obiettivo
Garantire la sicurezza dei dati e dei servizi offerti dall'app.

## Regole Operative
- Rate limiting sugli endpoint.
- Encryption end-to-end.
- Backup periodico dei database.

## Impatto sul codice
- Configurazione HTTPS obbligatoria.
- Logs di sicurezza centralizzati.
- Disaster Recovery configurato.

## Checklist
- [ ] HTTPS applicato.
- [ ] Backup configurati.
- [ ] Logs audit attivi.
